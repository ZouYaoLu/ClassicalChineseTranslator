# 智能文言文翻译器

一个基于 AI 大模型的文言文翻译工具，支持本地缓存和多种 AI 模型。

## 功能特点

- 🎯 **智能翻译**：支持通义千问、OpenAI 等多种大模型
- 💾 **本地缓存**：自动缓存翻译结果，提高响应速度
- 🔊 **语音朗读**：支持翻译结果的语音朗读功能
- 📚 **历史记录**：保存翻译历史，方便查看和复用
- ⚙️ **灵活配置**：可视化配置界面，轻松管理 AI 模型
- 🎨 **美观界面**：现代化的响应式设计

## 技术栈

- 后端：Python Flask
- 前端：HTML5 + CSS3 + JavaScript
- AI 模型：通义千问、OpenAI GPT
- 部署：支持宝塔面板一键部署

## 快速开始

### 本地运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 访问 `http://localhost:5000`

### 宝塔部署

详见 [部署文档.md](部署文档.md)

## 配置说明

首次使用需要配置 AI 模型的 API 密钥：

1. 访问配置管理页面
2. 选择要使用的 AI 模型
3. 填入对应的 API 密钥
4. 保存配置

## 使用方法

1. 在输入框中输入文言文
2. 点击"AI模型翻译"按钮
3. 等待翻译结果
4. 可以复制结果或点击朗读
