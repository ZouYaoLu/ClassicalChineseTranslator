# 智能文言文翻译器 - 宝塔面板部署文档

本文档详细说明如何在宝塔面板中部署智能文言文翻译器。

## 系统要求

- 宝塔面板 7.0 或更高版本
- Python 3.8 或更高版本
- 至少 1GB 内存
- 至少 1GB 可用磁盘空间

## 部署步骤

### 1. 安装 Python 项目管理器

1. 登录宝塔面板
2. 进入「软件商店」
3. 搜索并安装「Python项目管理器」插件

### 2. 创建网站

1. 进入「网站」→「添加站点」
2. 填写域名（例如：`wenyan.yourdomain.com`）
3. 选择 PHP 版本：纯静态
4. 创建数据库：不需要
5. 点击「提交」

### 3. 上传项目文件

1. 进入「文件」管理器
2. 导航到网站根目录（例如：`/www/wwwroot/wenyan.yourdomain.com`）
3. 上传整个 `wenyan-translator` 文件夹
4. 确保文件结构如下：
```
/www/wwwroot/wenyan.yourdomain.com/
├── app.py
├── requirements.txt
├── app/
├── static/
├── templates/
├── config/
└── cache/
```

### 4. 配置 Python 项目

1. 进入「Python项目管理器」
2. 点击「添加项目」
3. 填写配置：
   - 项目名称：文言文翻译器
   - 项目路径：`/www/wwwroot/wenyan.yourdomain.com`
   - 启动文件：`app.py`
   - 端口：5000（或其他未占用端口）
   - 启动方式：选择 Gunicorn

4. 点击「确定」创建项目

### 5. 安装依赖

1. 在 Python 项目管理器中找到刚创建的项目
2. 点击「模块」→「安装模块」
3. 选择「requirements.txt」方式安装
4. 等待安装完成

### 6. 配置反向代理

1. 进入「网站」→ 找到您的网站 → 点击「设置」
2. 选择「反向代理」标签
3. 点击「添加反向代理」
4. 填写配置：
   - 代理名称：flask_app
   - 目标 URL：`http://127.0.0.1:5000`
   - 发送域名：`$host`
5. 点击「提交」

### 7. 配置 SSL（可选但推荐）

1. 在网站设置中选择「SSL」标签
2. 选择「Let's Encrypt」免费证书
3. 勾选域名，点击「申请」
4. 申请成功后，开启「强制 HTTPS」

### 8. 启动项目

1. 返回 Python 项目管理器
2. 找到文言文翻译器项目
3. 点击「启动」按钮
4. 查看状态是否为「运行中」

### 9. 配置 AI 模型

1. 访问 `https://wenyan.yourdomain.com`（或您的域名）
2. 点击页面底部的「配置管理」
3. 配置至少一个 AI 模型：

#### 配置通义千问（推荐）
- 获取 API Key：
  1. 访问 [阿里云控制台](https://dashscope.console.aliyun.com/)
  2. 注册/登录账号
  3. 进入「API-KEY管理」
  4. 创建新的 API Key
- 在配置页面：
  1. 勾选「启用此模型」
  2. 填入 API 密钥
  3. 选择模型（推荐 qwen-max）
  4. 保存配置

#### 配置 OpenAI（可选）
- 获取 API Key：访问 [OpenAI平台](https://platform.openai.com/api-keys)
- 填写 API 密钥和相关配置

### 10. 设置开机自启

1. 在 Python 项目管理器中
2. 找到项目，点击「设置」
3. 开启「开机自启」选项

## 常见问题

### 1. 项目无法启动

- 检查端口是否被占用
- 查看项目日志：在项目管理器中点击「日志」
- 确保 Python 版本 ≥ 3.8

### 2. 翻译失败

- 检查 API 密钥是否正确
- 确保服务器可以访问外网
- 查看 `/cache/translations.json` 文件权限

### 3. 页面无法访问

- 检查反向代理配置
- 确保防火墙开放了相应端口
- 查看网站错误日志

### 4. 修改配置后不生效

- 在 Python 项目管理器中重启项目
- 清除浏览器缓存

## 性能优化建议

1. **启用缓存**：在配置页面确保「启用翻译缓存」已勾选
2. **使用 Redis**（可选）：安装 Redis 可提升缓存性能
3. **配置 CDN**：对静态资源启用 CDN 加速
4. **定期清理缓存**：当缓存文件过大时，可适当清理旧数据

## 备份建议

定期备份以下文件：
- `/config/config.json` - 配置文件
- `/cache/translations.json` - 翻译缓存

## 更新维护

1. 备份配置和缓存文件
2. 上传新版本文件
3. 在 Python 项目管理器中重启项目
4. 测试功能是否正常

## 技术支持

如遇到问题，请检查：
1. 项目日志（Python项目管理器 → 日志）
2. 网站错误日志（网站设置 → 网站日志）
3. 浏览器控制台错误信息
